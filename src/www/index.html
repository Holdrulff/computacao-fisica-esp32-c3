Static File Serving Demo



Este é um servidor web hospedado em uma placa de desenvolvimento ESP32 supermini com display OLED embutido.

Mais informação sobre o projeto básico em https://gitdub.com/FNakano/ProjetoBasico-CFA
As rotas que podem ser acessadas no servidor são:
rota efeito no navegador efeito no dispositivo
/ mostra a página index.html -
/hello mostra o texto 'hello from dv01.local' -
/storage mostra JSON com informações de armazenamento (total, usado, livre em MB e bytes, porcentagem usada) -
/led mostra o texto on caso o LED esteja apagado, off caso o LED esteja aceso -
/led/on mostra o texto on apaga o LED
/led/off mostra o texto off acende o LED
/led/blink?count=3&interval=0.5 mostra JSON com ação, contagem e intervalo pisca o LED 3 vezes com intervalo de 0.5s (parâmetros: count=1-20, interval=0.1-2.0)
/morse?text=SOS mostra JSON com texto, código Morse e duração pisca o LED em código Morse e mostra cada letra progressivamente no display conforme é transmitida (parâmetros: text=1-20 caracteres obrigatório, speed=0.1-0.5 segundos opcional)
/message mostra o texto que está sendo mostrado no display -
/message?text="xxx" mostra o texto passado como argumento em 'text' mostra no display o texto passado como argumento em 'text'
/www/webrepl/webrepl.html Mostra a página para acesso a interface de comando (REPL) para conectar, substituir o número da porta de 5000 para 8266 senha: star


O aplicativo é iniciado com o comando `import startsystem` digitado no REPL (tdonny).
